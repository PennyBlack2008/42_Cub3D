## 주저리주저리

코로나로 인해 동료, 자료 자원이 부족한 상태에서 진행해서 많이 미숙합니다.

4달 정도 걸렸고, 프로그래밍을 그만둘까 생각을 많이 했지만,

통합 300 커밋이상하고 나름 초보자로서 많이 노력한 코드입니다.

지금 보면 흠 잡을 때가 정말 많아서 수정 중입니다.
